# Лабораторная работа по ФП (Gleam)

## Подворный Артём Михайловчи P3313

### Вариант: 1, 30

## Описание задач
Реализованы два алгоритмических задания из проекта Эйлера:

1. **Задача 1**: Найти сумму всех чисел меньше заданного предела, кратных 3 или 5.
2. **Задача 2**: Найти сумму всех чисел, равных сумме пятых степеней своих цифр.

Для каждой задачи представлено несколько реализаций:
- рекурсивная (обычная и хвостовая),
- модульная реализация с использованием (`filter`, `fold`, `range`),
- с генерацией последовательности при помощи отображения (`map`)

## Ключевые элементы реализации

- Использованы **разные стили рекурсии**: простая и хвостовая (для TCO).
- Применены функции из стандартной библиотеки Gleam (`map`, `filter`, `fold`).
- Код разбит на **маленькие чистые функции**.
- Все функции **без побочных эффектов**, данные неизменяемы.

## Выводы

Писать на Gleam оказалось прикольно и интересно, особенно понравилилась реализация пайпов. Не понравилось отсутствие if-ов. На ЯП Gleam оказалось достаточно удобно реализовать все поставленные задачи.
